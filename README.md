# Awesomewm-dotfiles 😎
my awesomewm dotfiles :)

# Dependencies 🚀

- Awesomewm (window manager)
- Kitty (terminal emulator)
- Neovim (code-editor)
- Sarasa Mono font  
- Fish (shell)
- Picom (pijulius fork)
- Rofi (app launcher)
- Thunar (file mgr)
